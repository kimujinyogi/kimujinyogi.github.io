## ファイル・ディレクトリ規則

### 新規記事のパス構造

```
campcar/content/posts/<記事ディレクトリ名>/index.md
```

### ファイル命名規則

既存の記事から読み取れるパターン：

| プレフィックス | カテゴリ | 例 |
|---|---|---|
| `z` | 雑談・日常 | `z20260319.md` |
| `b` | 本のレビュー | `b20250507.md` |
| `mc` | マインクラフト | `mc20210725.md` |
| `campcar_` | キャンピングカー | `campcar_20200806.md` |
| `parenting_` | 育児 | `parenting_20240819.md` |
| `dev` | 開発 | `dev20240228.md` |
| `gm` | ゲーム | `gm20230606.md` |
| `ik` | 育児記 | `ik20230730.md` |
| `st` | 雑談/その他 | `st20201203.md` |

日付フォーマット: `YYYYMMDD`（今日の日付を使う）

同じ日に複数記事がある場合は末尾に `_0`, `_1` などを付ける。

## ショートコード

### 画像（中央揃え）

```
{{< imagelink src="/記事ディレクトリ名/image.jpg" class="imagecenter" title="キャプション" >}}
```

### サムネイル付き画像（拡大プレビュー）

```
{{< thumblink src="/記事ディレクトリ名/image.jpg" thumbnail="/記事ディレクトリ名/image_t.jpg" class="imagecenter" title="キャプション" >}}
```

- `src`: フル解像度画像
- `thumbnail`: サムネイル（`_t.jpg` または `_b.jpg` サフィックスが多い）

### YouTube 埋め込み

```
{{< youtube VIDEO_ID >}}
```

### 画像ファイルの配置場所

```
campcar/static/<記事ディレクトリ名>/image.jpg
```

## 記事作成の手順

ユーザーから記事作成を依頼されたら、以下の手順で進める：

1. **情報収集**: タイトル、カテゴリ、内容の概要をユーザーに確認する（不明な場合）
2. **ファイル名決定**: 上記の命名規則に従って `<ディレクトリ名>` を決定する
3. **Front matter 生成**: 適切な category, tags, series を設定する
4. **本文作成**: ユーザーの指示に従って日本語で記事を書く
5. **ファイル作成**: `campcar/content/posts/<ディレクトリ名>/index.md` に書き込む

---

## 文章スタイル

- **語り口**: 話し言葉に近い自然な日本語（ですます調）
- **トーン**: フレンドリー、個人的な体験談ベース
- **長さ**: 短くても OK（数百字の記事も存在する）
- **構成**: 小見出し（##, ###）を使って読みやすく整理
- **絵文字**: 基本的に使わない

---

## 注意事項

- `themes/zzo/` は Git サブモジュールなので直接編集しない
- CSS のカスタマイズは `campcar/assets/css/custom.css` で行う
- テンプレートのオーバーライドは `campcar/layouts/` 以下に配置
- `master` ブランチは CircleCI が自動生成するので直接編集しない
- `campcar` ブランチへの push でデプロイが走る
